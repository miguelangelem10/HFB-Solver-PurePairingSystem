# HFB-Solver-Pure Pairing System
Implementación numérica del método HFB para el modelo de pairing.
# Resolución Autoconsistente HFB en Sistemas de Muchos Cuerpos

Este repositorio contiene los códigos numéricos desarrollados en MATLAB para mi Trabajo de Fin de Grado (TFG) titulado "Estudio y aplicación de la transformación de Hartree-Fock-Bogoliubov compleja en sistemas cuánticos de muchos cuerpos".

## Contenido del Repositorio
* `EstructuraHFB.m`: Solucionador HFB con restricciones aplicado al modelo analítico de puro *pairing*.

## Requisitos
* MATLAB (versión R2020a o superior).
